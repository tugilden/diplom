# диплом

Алгоритм Мегиддо для решения задачи 2D линейного программирования с нахождением целочисленных решений.

## Структура

```
Megiddo/
├── main.py              # Запуск: Мегиддо → преобразование → целочисленные точки
├── megiddo_optimized.py # Алгоритм Мегиддо O(n) 
├── visualize.py         # Интерактивная визуализация (Plotly)
├── grahem.py            # Метод перечисления вершин
├── transform_new.py     # Преобразование неравенств
└── input2.txt           # Входные данные
```

## Входные данные

Файл (input2.txt):
- Строка 1: `p q` — коэффициенты целевой функции max p*x + q*y
- Строки 2+: `a b c` — ограничения a*x + b*y ≤ c
